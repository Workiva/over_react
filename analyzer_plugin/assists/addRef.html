addRef

Maturity: stable

Since 1.0.0

View the Project on GitHub workiva/over_react

When an over_react component builder is selected by the user, the assist adds a ref prop value on the builder, set equal to a variable that is added within the component definition, assigned to the return value of a call to createRef() for class-based components, or the return value of a call to useRef() for function components.
